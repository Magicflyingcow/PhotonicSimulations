Connection

The C13016 is a USB2/WinUSB device, not a USB serial device. This page uses an experimental reverse-engineered direct WebUSB capture path.
Direct WebUSB needs Chrome or Edge and OS permission to the device.

Acquisition

Direct WebUSB is experimental. The diagnostics parse little-endian image data using the device header framing. Average frames smooths the displayed spectrum by averaging the last N raw frames without changing exposure time.

Wavelength calibration

Wavelength is calculated as A0 + B1·pix + B2·pix² + B3·pix³ + B4·pix⁴ + B5·pix⁵. Use the coefficients supplied with your C12880MA if available.

Heart Rate monitor mode

— BPM
Enable this mode, start live acquisition, and hold a fingertip steadily in the 655 nm optical path.
Confidence: — Window: 0.0 s
The monitor ratios 652–658 nm against neighbouring red bands and the broad 600–700 nm intensity, then detrends the result. This rejects ambient changes, lamp flicker, exposure drift, and other common intensity fluctuations before estimating the dominant pulse frequency.
